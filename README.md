# 간단한 웹사이트 스타터

## 파일 구성
- `index.html`: 페이지 내용
- `assets/style.css`: 디자인(색, 폰트, 여백 등)
- `assets/script.js`: 간단한 동작(예: 연도 자동 반영)
- `images/`: 이미지 넣는 폴더

## 빠르게 수정하기
1) `index.html`에서 `<title>`과 `<h1>`, 소개 문장을 바꾸세요.
2) 연락처 이메일을 본인의 주소로 바꾸세요.
3) 필요 없는 섹션은 지우고, 텍스트를 원하는 내용으로 교체하세요.

## 배포 팁
- 별도의 프로그램 설치 없이 Netlify Drop에 폴더를 드래그앤드롭하면 바로 주소(URL)가 생깁니다.
